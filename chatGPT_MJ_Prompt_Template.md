# AI Prompt Challenge Format (Strict)

## Core Rules
- Each entry must follow this exact structure:
  1. Prompt (single paragraph, MidJourney-ready)
  2. Blank line
  3. Footer (3–4 lines)
  4. Separator line: ---

- The **daily phrase MUST appear at the very beginning of the prompt**, followed by a comma.

- Prompts should be:
  - Tight and MidJourney-optimized (no fluff)
  - Comma-separated descriptors
  - Cinematic, visual, descriptive
  - No quotation marks

- **Profile string behavior:**
  - Determined by the FIRST prompt in the chat
  - Must remain consistent for all prompts in that chat
  - Do NOT introduce or change unless explicitly told

- Footer must always follow this exact order:
  Line 1: [Month] [Day Number]  
  Line 2: [Daily Phrase]  
  Line 3: @[username]  
  Line 4: Hashtags (single line)

- Hashtag rules:
  - Must include:
    - Challenge tag (e.g., #NightmareArchitect)
    - #collabcrew (ALWAYS at the end of the tag list)
  - Include 4–6 additional relevant tags based on the prompt
  - Keep all hashtags on ONE line

---

## Output Template

[Daily Phrase], [visual prompt, comma-separated descriptors, cinematic, ultra-detailed] --profile [profile_string]

[Month] [Day]
[Daily Phrase]
@[username]
#[ChallengeTag] #[RelevantTag1] #[RelevantTag2] #[RelevantTag3] #[RelevantTag4] #[RelevantTag5] #collabcrew

## Enforcement Notes
- Never omit the daily phrase at the start of the prompt
- Never split hashtags across lines
- Never place footer before the prompt
- Never change profile string mid-chat unless instructed
- Keep prompts visually strong but concise
