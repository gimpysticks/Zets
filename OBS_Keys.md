# OBS Studio Hotkeys

## Streaming
- **F7** - Start Streaming
- **F8** - Stop Streaming

## Recording
- **F9** - Start Recording
- **F10** - Stop Recording
- **Ctrl+F9** - Pause Recording
- **Ctrl+F10** - Unpause Recording

## Replay Buffer
- **F5** - Start Replay Buffer
- **F6** - Stop Replay Buffer
- **Ctrl+F6** - Save Replay Buffer

## Virtual Camera
- **F1** - Start Virtual Camera
- **F2** - Stop Virtual Camera

## File Management
- **F3** - Split File
- **F4** - Add Chapter Marker

## Scene & Display
- **F12** - Transition
- **F11** - Screenshot
